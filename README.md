# 💧 Drink Water Reminder

A simple Python application that sends desktop notifications every hour to remind you to drink water and stay hydrated.

## Features

* Hourly desktop notifications
* Lightweight and easy to use
* Runs continuously in the background
* Cross-platform notifications using Plyer

## Requirements

* Python 3.x
* Plyer

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/prashantmanisrivastava2004-sudo/drink-water-reminder.git
cd drink-water-reminder
```

### 2. Install dependencies

```bash
pip install plyer
```

## Usage

Run the script:

```bash
python water_reminder.py
```

The program will send a desktop notification every hour reminding you to drink water.

## Example Notification

**Title:** Water Reminder

**Message:** It's time to drink water! Stay hydrated.
